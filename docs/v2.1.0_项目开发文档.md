# v2.1.0 项目开发文档

## 本版定位

`v2.1.0` 当前先推进“配置状态可视化 + 串口配置增强”。

相对于 `v2.0.0`，这版重点不是再扩展 NVS 基础能力，而是把已有配置系统做得：

- 更容易看见
- 更容易确认
- 更容易测试

## 当前已完成内容

### 1. LCD 配置来源显示

`display_service` 现在会在项目区显示：

```text
CFG : DEFAULT / NVS / MIXED / RUNTIME
```

这样上板时不用只看日志，也能快速知道当前配置是从哪里来的。

### 2. 串口命令增强

`config_cli_service` 已新增：

- `cfg status`
- `cfg source`
- `cfg test http`
- `cfg test ota`

### 3. OTA 手动检查入口

`ota_service` 已增加：

- `ota_service_check_now()`

这让 CLI 可以在当前网络已就绪时，主动触发一次 OTA 检查。
