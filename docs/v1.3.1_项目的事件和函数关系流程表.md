# v1.3.1 项目的事件和函数关系流程表

## 1. 版本定位

`v1.3.1` 在 `v1.3.0` 的基础上，继续完善 `XL9555` 板载交互能力。  
这一版的重点是：

- `KEY0 ~ KEY3` 四键完整接入
- 新增 `BEEP` 输出反馈
- 保持统一事件结构与任务通信主链不变
- 新增功能键与蜂鸣器测试模式

## 13. beep_service 脉冲时序图

这一块帮助理解为什么蜂鸣器没有在 `app_event_task` 里直接 `delay`，而是交给 `beep_service_process()` 周期推进。

### 13.1 短响时序图

```mermaid
sequenceDiagram
    participant EVT as app_event_task()
    participant BEEP as beep_service_play(SHORT)
    participant PROC as beep_service_process()
    participant BSP as bsp_xl9555_beep_set()

    EVT->>BEEP: pattern = SHORT
    BEEP->>BEEP: remaining_pulses = 1
    BEEP->>BEEP: on_ms = APP_BEEP_SHORT_ON_MS
    BEEP->>BSP: 输出打开
    loop 主循环周期推进
        PROC->>PROC: 判断是否到 on_ms
    end
    PROC->>BSP: 输出关闭
    PROC->>PROC: sequence_active = false
```
