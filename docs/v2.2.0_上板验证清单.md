# v2.2.0 上板验证清单

## 基础启动

- 启动日志版本显示为 `v2.2.0`
- 阶段名显示为 `v2.2.0 Button Menu Template`
- LCD 首页正常显示

## 菜单入口

- `KEY3` 长按可进入菜单
- 进入菜单后首页常规区域刷新暂停
- 再次按键时菜单不会误触发 LED / BEEP 原业务逻辑

## 菜单导航

- `KEY0` 可切换上一页或上一动作
- `KEY1` 可切换下一页或下一动作
- `KEY2` 可确认 / 执行动作
- `KEY3` 可返回上一层或退出菜单

## 菜单页面

- `MENU HOME` 能显示版本 / CFG / WIFI
- `MENU CONFIG` 能显示 `SSID / HTTP / OTA`
- `MENU ACTION` 能显示 `RESET_CFG / RELOAD_CFG / REBOOT`

## 菜单动作

- `RESET_CFG` 执行后配置恢复默认值
- `RELOAD_CFG` 执行后重新加载当前配置
- `REBOOT` 执行后设备正常重启

## 退出恢复

- 退出菜单后 LCD 首页恢复正常刷新
- 网络、HTTP、OTA 状态显示恢复正常
