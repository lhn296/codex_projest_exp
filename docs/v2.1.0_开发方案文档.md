# v2.1.0 开发方案文档

## 版本主题

配置状态可视化与串口配置增强版

## 版本目标

- 在 LCD 上显示当前配置来源摘要
- 增强串口配置命令，方便快速排查配置状态
- 增加 `cfg test http` 和 `cfg test ota`，支持手动测试配置有效性
- 保持现有 `config_service` / `config_cli_service` / `wifi_service` / `http_service` / `ota_service` 分层不变

## 主要实现点

### 1. 配置来源可视化

- `config_service` 继续维护：
  - `wifi_source`
  - `http_source`
  - `ota_source`
- 对外提供：
  - `config_service_get_*_source()`
  - `config_service_get_source_summary()`
- `display_service` 在项目区新增：
  - `CFG : DEFAULT / NVS / RUNTIME / MIXED`

### 2. 串口命令增强

在 `config_cli_service` 基础上新增：

- `cfg status`
- `cfg source`
- `cfg test http`
- `cfg test ota`

### 3. OTA 手动检查入口

- `ota_service` 增加 `ota_service_check_now()`
- 供 CLI 在联网成功后主动触发一次 OTA 检查
