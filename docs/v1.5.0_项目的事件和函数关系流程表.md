# v1.5.0 项目的事件和函数关系流程表

## 1. 版本定位

`v1.5.0` 是当前项目正式进入联网主线的第一个版本。  
这一版最重要的不是 HTTP 或 OTA，而是先把：

- `Wi-Fi STA`
- 网络状态管理
- 串口与 LCD 联动显示

这条最基础的联网链做稳。

## 9. Wi-Fi 状态模型图

```mermaid
flowchart TD
    A["WIFI_IDLE"] --> B["WIFI_CONNECTING"]
    B --> C["WIFI_CONNECTED"]
    C --> D["WIFI_GOT_IP"]
    B --> E["WIFI_DISCONNECTED"]
    C --> E
    D --> E
    E --> B
    E --> F["WIFI_ERROR"]
```

说明：

- `CONNECTED` 表示已经连上路由器
- `GOT_IP` 表示网络真正可用了
- 后面发 `HTTP` 请求时，通常应该等到 `GOT_IP`
